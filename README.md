
#Stopwatch
----------------------------------
Frontend skill test.

This is a simple web based stopwatch built with HTML,CSS, Bootstrap and vanilla Js.
## Features

- This has 4 button Start, Stop, Reset and Lap.

## Installation
Simply open the HTML file in a modern browser, or use Live Server on vscode.
